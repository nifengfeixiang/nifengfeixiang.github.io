1. Definition

子模性有很大的应用场景，如viral marketing，information gathering，image segmentation，document summarization等等；子模性是set function的一种性质，$f(S),S\subseteq V$；其实本质上是一种边际效益递减的函数（体现了一种diminishing return property）；首先我们定义discrete derivative:

Definition (Discrete derivative) : For a set function f : $2^V, S \subseteq V$ , and $e \in V$ , let $\Delta f (e | S) := f(S \cup \{e\}) − f(S)$ be the discrete derivative of $f$ at $ S$ with respect to $e$.

根据这个我们给出子模性的定义：

Definition (submodularity) A function $f$ is submodular if for every $A\subseteq B\subseteq V$, and $e\in V\setminus B$ it holds that: $\Delta(e|A)\ge \Delta(e|B)$; or for every $A,B\subseteq V$,$f(A \cap B)+f(A \cup B) \leq f(A)+f(B)$ ;

在子模函数中，有一种重要的子类型为单调的子模函数monotone：如果$A\subseteq B\subseteq V$，$f(A)\le f(B)$；

同时，子模函数也具有很多有用的性质：

  • 多个子模函数的非负线性组合依然具有子模性；
  • 如果采用residual（剩余），依然是子模函数：如$g$ 是子模函数，$A,B\subset V$，那么$f(S)=g(S\cup B)-g(B)$依然是子模函数；
  • 如果是使用truncation，与一个常数$c$取最大最小，$f(S)=\min\{g(S),c\}$依然保证子模性；需要注意的是，多个子模函数取max或者是min，这样的函数不一定具有子模性；

  • Knapsack constraints：每个item可能有成本参数$c_i$，需要在budget $B $的限制下最大化函数；
